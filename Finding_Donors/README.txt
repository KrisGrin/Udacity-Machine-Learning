This is the third assignment submitted to Udacity towards a Machine Learning Engineer Nanodegree. 
The assignment focused on predicting finding donors who's income is greater than 50K, using UCI's Machine Learning Repository.
See: https://archive.ics.uci.edu/ml/datasets/Census+Income
The focus of this assignment was on:
1. Transforming and normalizing data;
2. Model selection via testing the performance of various models; 
3. Improving model results by tuning parameters, using GridSearch; and,
4. Feature selection.
